Para instalar nltk

* python.exe -m pip install -U nltk

Para descargar stopwords, desde una ejecución de python
*
    import nltk
    nltk.download("stopwords")
    exit()

Para instalar pandas
* python.exe -m pip install -U pandas

Para instalar matplotlib
* python.exe -m pip install -U pip setuptools
* python.exe -m pip install -U matplotlib

Para instalar word_cloud

* python.exe -m pip install wordcloud
    - Si la instalación falla, probar con Archivos de instalación, copiándolos a la ruta de instalación de Python, y ejecutar
        * python.exe -m pip install wordcloud-1.3.1-cp36-cp36m-win32.whl
        ó
        * python.exe -m pip install wordcloud-1.3.1-cp36-cp36m-win_amd64.whl
